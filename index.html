GraphRAG for Biology

GraphRAG-Biology is a specialized implementation of Microsoft's GraphRAG framework, tailored for processing and analyzing biological research papers.

This project leverages the structured, hierarchical approach to Retrieval Augmented Generation (RAG) provided by GraphRAG to enhance the analysis of complex biological information.

TLDR Science: Research Paper Summarizer

Project that aims to automate the summarization of scientific research papers into concise, easy-to-understand summaries sent directly to me via Whatsapp. Born out of a need to efficiently parse through autoimmunity research, TLDR Science leverages AI and programming to help focus on the big picture.

The project includes a Chrome extension for easy text extraction, a Python backend for processing, and uses OpenAI GPT-4 for analysis. It generates PDF summaries and sends them via WhatsApp, streamlining the research review process.

Project Generator: The multi-agent AI Project Generator

A web application that leverages multi-agent AI workflow to automatically generate project structures, code, and documentation based on user-provided objectives. This tool streamlines the project initialization process, saving time and providing a solid starting point for various coding projects.

The best part is that all the code generated by the agents is output as code files. With a simple prompt, an entire project is coded and given to the user ready.
